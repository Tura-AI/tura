# Personas

Personas control Tura's user-facing voice, communication style, and optional
avatar/expression metadata. They do not grant tools or change provider routing.

The full reference is [docs/core/personas.md](../../docs/core/personas.md).
Customization instructions are in [Custom personas](../customization/custom-personas.md).

## Boundary

- GUI sessions load persona identity plus rich communication style.
- CLI sessions use the shared CLI communication style.
- Agents still own tools and model behavior.
