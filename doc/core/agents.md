# Agents

Agents are Tura runtime work profiles. They select prompt resources, command
capabilities, provider/model route preferences, operation-manual policy, and
reporting/validator flags.

The full reference is [docs/core/agents.md](../../docs/core/agents.md).
Customization instructions are in [Custom agents](../customization/custom-agents.md).

## Boundary

- Agent: tools, model defaults, prompt resources, operation manual policy.
- Persona: communication style and visible identity.
- Runtime prompt: task-specific operating discipline.
- Provider config: concrete provider/model/auth routing.
