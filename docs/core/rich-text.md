# Rich text

Tura uses a restricted HTML-flavored rich text protocol for normal interactive
messages and a separate plain terminal style for CLI sessions.

The full reference is [HTML rich text](html-rich-text.md).

## Why this page exists

The GitBook navigation uses the shorter label "Rich text". The detailed owner
document keeps the implementation-specific title because it explains the exact
HTML subset, media tokens, emoji reaction/sticker tokens, and GUI/TUI renderer
behavior.
