# Dynamic prompt injection

Dynamic prompt injection is Tura's runtime-owned prompt assembly layer. It adds
state-appropriate prompt fragments for persona, task type, retry state,
compaction pressure, and command capabilities without letting user text become
privileged policy.

The full reference is [Prompt style and dynamic prompt injection](prompt-style.md).
