# Aegis Rift Starship HUD

Standalone Three.js demo for a cinematic 3D starship game capture with tactical HUD.

## Assets

- `starship.glb`, `wingman.glb`, `enemy_large.glb`, `planet.glb`, `base_large.glb`: Quaternius Ultimate Space Kit via Poly Pizza, Public Domain CC0.
- `starfield.jpg`: NASA Scientific Visualization Studio, "An Elsewhere Starfield".

The page uses local downloaded assets from `./assets` and loads Three.js/lucide from public CDNs.

## Run

```powershell
python -m http.server 4177
```

Open `http://127.0.0.1:4177/` from this directory.
